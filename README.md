This is a MERN stack based freelancing platform where clients can post jobs and freelancers can apply for them.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Features
- User authentication
- Job posting
- Job application
- Role-based access

## How to Run
1. Install dependencies in client and api folders
2. Start backend using npm start
3. Start frontend using npm start
